환경 MBTI
요약
  • 기획의도
    여러 산업이 발달하고, 고도화되면서 인간의 삶은 윤택해졌지만 지구 곳곳에서는 여러 환경문제가 발생하고 있습니다.
    수많은 동식물들이 지구상에서 사라지고, 이상기후로 많은 사람들이 죽어가고, 인간은 점점 설 곳을 잃어가지만 많은 사람들은 이를 알 지 못합니다.
    이 프로젝트는 환경에 관심이 많은 두 사람이 모여 이러한 문제를 해결하고자 만든 프로젝트입니다.
    환경문제는 아직 사람들의 관심이 많이 부족한 만큼 경각심을 주는 것과 더불어 재미요소가 필요하다는 생각이 들었고, 이를 위해 하나의 SNS 컨텐츠로 자리잡은 MBTI와 환경을 접목시켜보았습니다.
  • 개발기간
    2022.09.17 - 2022.09.25
  • 소스코드
    환경 mbti 깃허브 바로가기
  • 도메인
    환경 mbti 사이트 바로가기
  • 멤버
    팀 프로젝트(2인)
개발목표
- 공공데이터 포털의 Open API 이용
  • TM 좌표 기반의 가장 가까운 관측소 거리 계산, 관측소 정보 조회
  • 관측소 정보 data를 파싱해 해당 관측소의 현재 대기상태 조회
- JSON, XML 파일을 파싱하여 이용자에게 데이터 제공
- rest API
  • Kakao 공유하기 기능을 사용 해 mbti 페이지 다른 사용자에게 공유
  • kakao 지도 api를 이용한 좌표계 변환(위/경도 - TM 좌표)
  • naver 공유 api를 이용한 사이트 공유
- html2canvas를 사용하여 mbti 결과 화면 캡쳐 후 데스크톱 download 폴더에 저장
- geolocation api를 활용해 사용자의 지리 정보 조회하기
- flex, grid를 활용한 모바일 화면 지원
- SPRING MVC를 활용한 프로젝트 구조 단순화
이슈
  • 위치조회를 위한 https 보안 프로토콜 도입이 필요
주요기능 (기능을 클릭하면 자세히 볼 수 있어요)
  • mbti 문제 풀고 결과 보기
    기능 : 사용자 정보 입력하기, mbti 문제 불러오기, 사용자 정답 저장하기, 결과생성하기, mbti 결과 불러오기
    팀원이 해당 프로젝트에 필요한 기본적인 기능을 구현하기 보다는 새로운 api를 적용하는 것에 열정이 많아 대부분의 기능을 혼자 구현하게 되었습니다.
    spring mvc 프레임워크를 바탕으로 view, controller, model을 각각 나누어 개발하는 경험을 하지 못한 점, 기능에 대한 의논을 많이 하지 못해 더 나은 코드를 작성하지 못한 점이 많이 아쉬웠습니다.
    그러나 여러기능을 개발하며 spring mvc 프레임워크를 더 잘 이해할 수 있었고, 저의 부족한 점을 돌아볼 수 있었습니다.
  • 대기오염 정보 조회하기
    사용자의 위치와 가장 가까운 관측소에서 측정한 대기정보를 조회
    - geolocation를 활용해 사용자의 지리정보(x, y) 획득

    - kakao local 지도 rest api를 활용해 획득한 좌표를 TM좌표계의 좌표로 변환

    - 관측소 정보 open api의 조회 URL에 TM 좌표값을 파라미터로 전송해 가장 가까운 관측소의 정보를 얻음

    - 관측소 정보를 파싱해 관측소 이름을 얻은 후 인코딩
    - 대기정보조회 open api를 이용해 해당 관측소의 대기정보를 Json타ㄹ입으로 얻은 후 데이터 파싱
    • 조회하는 데이터는 많고, 사용자의 위치정보를 이용하는 만큼 보안과 속도 사이에서 고민을 많이 하였으나,
      프로젝트의 특성과, 위치정보를 얻지 못할 경우가 많을 것을 예상하여 비교적 속도가 빠른 json 타입의 데이터를 파싱하엿음
    위치정보제공에 동의하지 않은 사용자나 브라우저의 버전에 따라 보안 프로토콜에서만 api가 동작하는 경우에는 사용자의 지리정보를 받아오지 못하는 것이 아쉬웠습니다.
    대안으로 사용자 위치정보를 얻지 않고 임의의 위치를 지정하였으나,
    보안에 대비하거나, 검색, 차세대 기능들을 도입할 경우를 대비해 HTTPS 도입이 필요함을 실감하게 되었습니다.
  • 오존 경보 조회하기
    - 공공데이터 포털의 open api를 이용해 한 달 동안의 전국 오존 경보를 XML 타입으로 조회
    - 조회한 XML 데이터를 파싱 후 view로 전송
  • 결과 공유하기
    기능 : 네이버 공유하기, 카카오톡 친구에게 공유하기
    - 자바스크립트를 이용해 naver 공유하기 기능
    - 카카오톡 공유 rest api를 사용해 공유하기 기능 제공
    • html2canvas를 사용하여 mbti 결과 화면 캡쳐 후 데스크톱 download 폴더에 저장
    • 저장한 이미지와 텍스트를 카카오톡 공유하기 템플릿에 맞게 구성
    • rest api를 통해 카카오톡 메시지 전송
