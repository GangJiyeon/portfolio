특히 스케줄을 효율적으로 관리하고, 관리하는 습관을 기를 수 있는 웹 사이트를 만들고자 노력하였습니다.
여러 기능을 추가하고, 프로젝트를 개선해나가면서 2023년이 되기 전 실제로 서비스하는 것을 목표로 하고 있습니다.

개발목표
- sns 로그인 api(naver, kakao)를 이용한 간편한 가입과 이용
- 갯수제한 없이 직관적인 사용이 가능한 투두리스트, 프로젝트 관리기능 제공
- jquery fullCalendar 플러그인 활용해 일정관리 기능을 시각적으로 제공
- sms 알림 rest api를 활용해 스케줄 작성 알림, 스케줄 체크 알림, 스케줄 알림 기능을 제공한 효율정인 일정관리

주요기능 (기능을 클릭하면 자세히 볼 수 있어요)
  • 회원관리
    전체기능 : 로그인, 로그아웃, 정보조회, 회원가입, 회원탈퇴, 아이디/비밀번호 찾기, sns 로그인, 본인인증
    - sns login rest api(naver, kakao)를 활용한 간편 회원가입
    • rest api의 난수 생성 전 데이터를 파싱해 등록되지 않은 계정일 경우 해당 사이트의 아이디 파라미터 값에 맞는 random_id 생성


    • 생성한 random_id와 sns_id를 sns테이블에 저장한 후, api에서 제공받은 회원정보와 random_id를 회원 테이블에 저장

    - kg inicis rest api를 활용한 간편 본인인증
    - session에 저장된 sns계정(kakao, naver, none)에 따른 아이디, 비밀번호 찾기 기능 제공
  • 투두리스트
    전체기능 : 투투리스트 조회/작성/수정/삭제, 투두리스트 아이템 삭제
    - 외래키를 사용해 TODOLIST_TB, TODOLIST_ITEM_TB 테이블을 참조시켜 데이터 중복을 최소화

    - ArrayList와 향상된 for문을 통해 참조테이블의 데이터를 조회한 후 뷰로 전송
    - c:if 태그와 jquery를 활용한 뷰 구성
  • 프로젝트
    전체기능 : 프로젝트 조회/작성/수정/삭제, 소프로젝트 삭제, 프로젝트 아이템 삭제
    - 외래키를 사용해 PROJECT_TB, PROJ_MID_TB, PROJ_ITEM_TB 테이블을 참조시켜 데이터 중복을 최소화

    - ArrayList와 향상된 for문을 통해 참조테이블의 데이터를 조회한 후 뷰로 전송

    - c:if 태그와 jquery를 활용한 뷰 구성
    - css 'overflow: scroll'를 활용해 div 요소 구성
  • 캘린더
    전체기능 : 일정 조회, 일정 수정, 일정 삭제, 일정 추가
    - jquery fullCalendar 플러그인 활용한 view 페이지 및 기능 구성
    • 커스덤 버튼을 활용한 헤더구성 및 일정 추가 기능 구현

  • 알림
    전체기능 : 사용자 알림 설정 조회, 알림 설정 수정, 스케줄 작성 알림, 스케줄 체크 알림, 일정 알림
    - naver cloud platform의 sms rest api를 활용한 sms 전송
    • 시그니처(공통 헤더 마지막 필드(x-ncp-apigw-signature-v2)에 들어가는 값) 생성
      -> 요청에 맞게 StringToSign을 생성하고 SecretKey로 HmacSHA256 알고리즘으로 암호화한 후 Base64로 인코딩
      -> 인코딩한 값을 x-ncp-apigw-signature-v2로 사용
    • 메시지 발송
      -> 요청 url 구성
      -> 요청 body 구성
      -> sms 메세지 발송

  • 게시판
    전체기능 : 전체/상세/내가 쓴 글 조회, 게시글 수정/삭제/등록, 조회수, 게시글/댓글 좋아요, 댓글 수정/삭제/조회/등록, 대댓글 작성/삭제
    - 전체 게시글 조회 페이지네이션 적용
    - content 높이를 일정하게 유지하기 위해 이미지의 가로 넓이를 측정한 후 최대 세로 길이 지정
    -> 이미지 세로 길이가 가로 넓이보다 짧은 경우 이미지 위 아래 여백을 줌
    -> croppers.js 플러그인을 사용해 이미지 편집 기능 제공 예정
